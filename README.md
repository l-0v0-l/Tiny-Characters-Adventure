# 五.程式介紹(含UML 圖、操作說明)
## 如何操作？
### 遊戲可選擇三隻角色其中一隻作為出戰角色
### 每一隻角色的技能及攻擊方式皆不同
### 使用**W A S D**進行移動
### 按下**SPACE**可以發射子彈
### 按下**K**可以發動技能(有冷卻)
### 挑戰失敗可以**提升素質**繼續挑戰
### 遊戲只有一關 但王很難打
